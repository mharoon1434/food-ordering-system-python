<h1>🍔 Food Ordering System (Python OOP Project)</h1>

<h2>📌 Overview</h2>
<p>
This project is a <b>console-based Food Ordering System</b> developed using <b>Python</b>.
It simulates a real-world food delivery platform (like Foodpanda) with multiple user roles such as
<b>Admin, Worker, Delivery Boy, and Customer</b>.
</p>

<p>The project demonstrates:</p>
<ul>
<li>Object-Oriented Programming (OOP)</li>
<li>File Handling (Data Persistence)</li>
<li>Modular Programming</li>
</ul>

<hr>

<h2>🚀 Features</h2>

<h3>👤 User Roles</h3>
<ul>
<li><b>Admin</b>
    <ul>
        <li>Add food items</li>
        <li>View all orders</li>
    </ul>
</li>

<li><b>Worker</b>
    <ul>
        <li>Manage food items</li>
    </ul>
</li>

<li><b>Delivery Boy</b>
    <ul>
        <li>View pending orders</li>
        <li>Mark orders as delivered</li>
    </ul>
</li>

<li><b>Customer</b>
    <ul>
        <li>Create account & login</li>
        <li>Order food</li>
        <li>View & modify orders</li>
    </ul>
</li>
</ul>

<hr>

<h3>🍕 Food Management</h3>
<ul>
<li>Add new food items</li>
<li>View available food menu</li>
</ul>

<hr>

<h3>📦 Order Management</h3>
<ul>
<li>Place orders</li>
<li>Update or cancel orders</li>
<li>Track delivery status (Pending → Delivered)</li>
</ul>

<hr>

<h3>💾 Data Persistence</h3>
<ul>
<li>Data is stored using <b>file handling</b></li>
<li>System restores data when program restarts</li>
</ul>
